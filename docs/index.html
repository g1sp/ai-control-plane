What it does

Every request and tool call is intercepted and evaluated before reaching the model or executing.

🔍

ML Threat Detection

TF-IDF cosine similarity classifier detects prompt injection, jailbreaks, PII exfiltration, credential theft, and indirect injection in under 5ms — no external APIs, fully deterministic.

📋

Policy-as-Code

YAML rules define what gets blocked, escalated, or allowed. Hot-reload without restart. Conditions use a sandboxed evaluator — no import, open, or os access from rule files.

🔐

Encrypted Audit Trail

AES-256-GCM encryption on all sensitive audit fields at write time. Decryption only at compliance endpoints — analytics and dashboards never see plaintext.

👤

Human-in-the-Loop

High-risk actions and expensive requests route to a review queue. Configurable timeout behavior defaults to deny — fail-safe when no operator is available.

⚡

Redis Rate Limiting

Sliding window rate limiter using Redis sorted sets. More accurate than fixed-window for burst protection. Falls back to in-memory automatically if Redis is unavailable.

🤖

Agent Orchestration

Multi-step agent executor with tool-calling — HTTP, code execution, search. Three-layer security: input validation, tool validators, approval queue. WebSocket streaming.
